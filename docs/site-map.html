Token-Gated Pages

Access requires a ?token= parameter in the URL. Tokens are generated at /internal/tokens. Without a valid token, visitors see a gate page.

/ekthesis Gated Business case / investor deck. Share with press, partners, and potential collaborators via generated token links.

URL format: atnocost.cc/ekthesis?token=XXXX
Gate page: /ekthesis/gate
Fallback token (env var): 6958 Token
/brief Gated Creative brief — team use only. Internal reference for collaborators, not for external distribution.

URL format: atnocost.cc/brief?token=XXXX
Gate page: /brief/gate Token · Team
